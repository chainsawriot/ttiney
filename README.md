# ttiney
A simple command line tool to tweet and toot at the same time.

## Set up

create a .env file with the following environment variables

TW_CONKEY = ""
TW_CONSEC = ""
TW_ACCTOK = ""
TW_ACCTOKSEC = ""
MAS_BASEURL = ""
MAS_LOGIN = ""
MAS_PASSWORD = ""

## Useage

Either

```{bash}
./ttiney
```

or 

```{bash}
./ttiney "I want to toot!"
```
